# Walkthrough: Agent Manager 導入完了

Agent Manager システム（V15）の導入が完了しました。

## 変更内容

### バックエンド (AI Agents)
- **ScoringAgent**: フロントエンドにあったスコアリングロジックを移植。
- **StrategyAgent**: レース展開の分析と、具体的な買い目（戦略）の構築を担当。
- **AgentManager**: 各エージェントを束ね、単一の予測結果を生成。

### API
- `/api/predict`: URLのスクレイピングから Agent Manager による予測までを一括で実行するエンドポイントを新設。

### フロントエンド
- **app_v15.js**: ロジックを排除し、`/api/predict` の結果を描画する軽量な最新版。
- **index.html**: V15仕様にアップデート。

## 検証結果

### ユニットテスト（ロジックの動作）
バックエンドでのスコアリング・戦略構築が正しく行われることを確認しました。

### UI表示
`index.html` から最新の `app_v15.js` が読み込まれ、バックエンドからの予測結果が正しくカード形式で表示されることを確認しました。

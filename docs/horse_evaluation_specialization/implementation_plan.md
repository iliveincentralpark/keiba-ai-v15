# 馬評価特化ツールへの刷新計画

AI競馬アプリを「買い目生成機」から、AIが馬の能力や妙味を評価する「馬評価特化型分析ツール」へと転換する。

## 変更の目的
- 買い目に従うのではなく、AIの分析根拠（実力・妙味・近走）を見てユーザー自身が判断できるようにする。
- 数値の羅列ではなく、競馬ファンが納得できる「自然な言葉」での解説を提供する。

## 主要な変更内容

### 1. UIの全面刷新
- **買い目機能の廃止**: 3連複などの買い目カード、軍資金入力、保存機能を削除。
- **馬評価カード**: 「本命」「対抗」「穴馬」を際立たせたカード形式で表示。実力・妙味・総合スコアをバーで可視化。
- **サマリーパネル**: 最上部にピックアップされた馬の一覧を表示し、一目で全体像を把握可能にする。
- **「あなたへのおすすめ」**: ユーザーの過去の的中履歴（DNA）に基づき、相性が良い馬をレコメンド。

### 2. AI解説エンジン（manager.py）
- 数値を述べるだけの「意味のわからない説明」を廃止。
- 「直近2連勝中」「上がり平均33.8秒と切れ足が際立つ」「○人気にしては大幅割安」など、実データに基づいた人間らしい評価文を生成。

### 3. スコアリングの高度化（scoring_agent.py）
- **近走成績の反映**: `shutuba_past.html` から全馬の直近5走の着順・上がりタイムを取得。
- **穴馬ロジックの修正**: 4着以下の馬にデータがない仕様によるバグを解消し、5人気以降の中穴・大穴を正しく捕捉。

## 完了判定
- GitHubへプッシュし、Render上のPWAで買い目が表示されず、馬評価と解説が正しく表示されること。

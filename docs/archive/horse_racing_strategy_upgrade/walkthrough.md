# 修正完了レポート：AI戦略買い目ロジックの導入とURL正規化の強化

## 実施した修正

### 1. AI戦略選定ロジックの刷新 (app.js)
単なる人気順の選定から、期待値を考慮した AI らしい選定アルゴリズムへ移行しました。
- **穴馬（激走馬）の自動抽出**: 6〜10番人気の中から、オッズ分布に「歪み」がある馬を AI が自動的にピックアップし、買い目に組み込みます。
- **荒れ判定との連動**: レースが「波乱・混戦」と判定された場合、穴馬の採用率を自動で調整します。

### 2. URL自動正規化の強化 (app.py)
特集ページURLなどから発生する「データの乖離・欠損」を根本解決しました。
- **正規URLへの自動変換**: 特集ページが入力されても、内部で自動的に「正規の出馬表URL」に変換して取得。
- **メリット**: 騎手名の文字切れ（ルメー...等）を防止し、常に最新かつ正確なオッズ・人気データを担保します。

## 動作確認方法
1.  バックエンドの起動: `python main.py`
2.  アクセス: `http://localhost:8000`
3.  スマホでの確認: PCのプライベートIP（例: `http://192.168.x.x:8000`）へスマホからアクセスすることで、PWAとして利用可能です。
